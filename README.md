
📊 Crowdfunding Data Analysis Project

📌 Project Overview

This project focuses on analyzing crowdfunding campaign data to understand the factors that contribute to project success or failure.
The analysis helps identify trends, patterns, and key drivers behind successful fundraising campaigns.

⸻

🛠️ Tools & Technologies
	•	Microsoft Excel
	•	Tableau
	•	Data Cleaning & Transformation
	•	Data Visualization

⸻

📂 Dataset Description

The dataset contains information about crowdfunding projects, including:
	•	Project Name
	•	Category & Sub-category
	•	Goal Amount
	•	Funds Raised (USD)
	•	Backers Count
	•	Launch Date & Deadline
	•	Project Status (Successful/Failed)

⸻

📊 Dashboard Preview
⸻

📈 Key Insights
	•	Projects with lower funding goals have a higher success rate
	•	Technology and Design categories attract more backers
	•	Campaigns with shorter duration tend to perform better
	•	Higher number of backers strongly correlates with project success
	•	Most successful campaigns are launched during specific months

⸻

🎯 Objectives
	•	Analyze success vs failure rate of crowdfunding campaigns
	•	Identify key factors affecting fundraising success
	•	Build interactive dashboards for data-driven insights

⸻

📊 Key Analysis Performed
	•	Success Rate Analysis
	•	Category-wise Performance
	•	Goal vs Funds Raised Comparison
	•	Time-based Trend Analysis
	•	Backers Impact Study

⸻

🚀 Future Improvements
	•	Predict project success using Machine Learning
	•	Add SQL-based analysis
	•	Create advanced Tableau dashboards with filters
